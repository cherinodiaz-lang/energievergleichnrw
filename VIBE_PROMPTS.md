```

1. HERO SECTION:

- Untertext: "Unabhängiger Vergleich von über 1.000 Strom- und Gastarifen. Kostenlos, schnell und unverbindlich."
- Call-to-Action Button (primär): "Jetzt vergleichen"
- Hintergrundbild: Moderne Windkraftanlagen oder Solaranlage in NRW

2. VORTEILE-SECTION (3 Spalten):
- Spalte 1: Icon Sparschwein + "Über 800€ sparen" + "Durchschnittliche Ersparnis beim Wechsel"
- Spalte 2: Icon Uhr + "In 3 Minuten vergleichen" + "Einfach PLZ eingeben und Tarife finden"

3. VERGLEICHSRECHNER:
- Formular mit Feldern: PLZ, Verbrauch in kWh, Tariftyp (Strom/Gas)

4. TRUST-SECTION:

- Accordion mit 5-7 häufigen Fragen

6. CLOSING CTA:
- Überschrift: "Bereit zum Wechseln?"

- Button: "Kostenlos vergleichen"

Design: Modern, clean, vertrauenswürdig. Farben: Blau (#2563eb) als Primärfarbe, Weiß, Grün-Akzente.
Mobile-First: Alles muss perfekt auf Smartphones funktionieren.
```

## 🔍 8. SEO TECHNICAL SETUP

```
Stelle sicher, dass folgende technische SEO-Elemente korrekt konfiguriert sind:

1. META TAGS:
- Jede Seite hat unique Title (max. 60 Zeichen)
- Jede Seite hat unique Description (max. 160 Zeichen)
- Open Graph Tags für Social Media
- Twitter Cards

2. SCHEMA MARKUP:
- Organization Schema auf Homepage
- LocalBusiness Schema auf City-Pages
- FAQPage Schema auf FAQ-Sections
- BreadcrumbList für Navigation

3. INTERNE VERLINKUNG:
- Sinnvolle Link-Struktur
- Breadcrumbs auf allen Unterseiten
- Related Pages am Seitenende

4. SITEMAP:
- XML Sitemap erstellen
- In Google Search Console einreichen
- Alle wichtigen Seiten inkludieren

5. ROBOTS.TXT:
- Crawler-Zugriff erlauben
- Admin-Bereiche blockieren
- Sitemap-Verweis

6. PAGE SPEED:
- Bilder komprimieren (WebP)
- CSS/JS minifizieren
- Browser-Caching aktivieren
- CDN nutzen (Wix macht das automatisch)
```
